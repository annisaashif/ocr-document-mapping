# TDS Parser

## Description
TDS Parser is a tool for parsing and processing TDS (Tabular Data Stream) protocol data.

## Features
- Parse TDS protocol messages
- Handle database communication
- Process tabular data streams

## Usage
```python
from tds_parser import Parser

parser = Parser()
# Your code here
```

## Requirements
- Python 3.8+
- Dependencies listed in `requirements.txt`
